
# Hugeicons Elementor Plugin

![Hugeicons library](https://cdn.hugeicons.com/blog/1707672477487-image.png)

## Description

The Hugeicons Elementor Plugin is your gateway to a vast universe of icons, offering seamless access to over **46,000 icons in 10 distinctive styles**. With three style categories—Rounded, Standard, and Sharp—this plugin is meticulously designed to integrate with Elementor, allowing you to find, customize, and embed premium quality icons directly into your pages. Whether you're designing a website, creating a project in Figma, or developing an app, Hugeicons has the versatility to elevate your design work.

## Widgets

The plugin includes **5 powerful widgets**:

| Widget | Description |
|--------|-------------|
| **Icon** | Display any icon from our library with full customization options including color, size, alignment, and link. |
| **Icon Button** | A customizable button with an icon. Position the icon before or after the text, with full styling control including hover states. |
| **Icon Box** | Combine an icon with a title and description. Perfect for feature sections with flexible icon positioning (top, left, or right). |
| **Icon List** | Create beautiful lists with icons. Each item can have its own icon, text, and optional link. Supports vertical and horizontal layouts with optional dividers. |
| **FAQ** | An accordion-style FAQ widget with expandable/collapsible answers. Includes customizable expand/collapse icons and optional question icons. |

## Installation

1. Download the Hugeicons Elementor Plugin from the WordPress Plugin Store.
2. Go to your WordPress dashboard, navigate to Plugins > Add New, and upload the plugin file.
3. Activate the plugin and enter your license key at Settings > Hugeicons Pro.

## Usage

1. Within the Elementor Editor, find the **Hugeicons** category in the widget panel.
2. Drag any of our widgets (Icon, Icon Button, Icon Box, Icon List, FAQ) to your page.
3. Click the icon field to browse our library of 46,000+ icons. Use the search function to find exactly what you need.
4. Customize your widget using the style options—adjust size, color, alignment, and more.
5. Publish your page and enjoy your beautiful icons!
